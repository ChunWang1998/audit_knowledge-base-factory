Introduction
This audit covers two minor changes to the Notional Exponent core code in addition to
an integration with the Inifnifi protocol.

• Adding enterWithYieldToken to AbstractLendingRouter.sol. Allows a user to swap a
leveraged position on the yieldToken from a non-Notional Morpho market into the
Notional version of that Morpho market. Callable only by whitelisted
positionSwapContract.

• Adding enterWithYieldTokenAndLeverage to AbstractLendingRouter.sol. Allows
users to enter a position by depositing the yieldToken instead of the borrow asset.

• Adds an Infinifi withdraw request manager. This contract enables siUSD and liUSD
yieldTokens.
